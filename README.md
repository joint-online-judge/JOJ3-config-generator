# This is the config file generator for the course ENGR1510J, ECE4820J, etc.

## Specification about the task name

- For homework, name it as: `hi exj`
  - `hi` represent the homework name
  - `exj` represent the exercise name
- For project, name it as: `pi mj`
  - `pi` represent the project name
  - `mj` represent the milestone name
