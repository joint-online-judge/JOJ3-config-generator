# JOJ3-config-generator

## Getting Started

1. Install [Python>=3.10](https://www.python.org/) and [PDM](https://pdm-project.org/).
2. Install deps by `pdm install --prod`.
3. Run it by `pdm run joj3-config-generator --help`

### For developers

```bash
pdm install --dev
pdm run pre-commit install
```
